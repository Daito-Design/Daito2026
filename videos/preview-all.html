Usage Instructions

Option 1: Use as HTML Background (Recommended)

Embed directly in the case study page using an iframe:

<div class="video-background">
    <iframe src="videos/shell-spatial-computing.html" frameborder="0"></iframe>
</div>

Option 2: Convert to MP4 Video

To create MP4 files for better performance and compatibility:

  1. Open the HTML file in Chrome at your desired resolution
  2. Use screen recording software (OBS, QuickTime, or Chrome DevTools)
  3. Record for 10-15 seconds (animations loop smoothly)
  4. Export as MP4 with H.264 codec

Option 3: Command Line Recording (Advanced)

Use headless Chrome with ffmpeg:

# Install puppeteer-screen-recorder
npm install puppeteer puppeteer-screen-recorder

# Or use Chrome's native recording
google-chrome --headless --screenshot --window-size=1920,1080 shell-spatial-computing.html

Customization

Each animation is fully customizable. Edit the JavaScript in each HTML file to adjust:

  • Colors: Change the color constants at the top
  • Speed: Adjust animation speed values
  • Density: Change particle count and grid spacing
  • Effects: Add/remove visual elements
